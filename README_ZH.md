## 项目说明

---

按照vuepress官方文档搭建的博客。主要难度在于配置文件config.js，其他方面按照文档来总没错。谢谢提供这么优秀的框架。



## 快速上手

---

* 运行
```js
npm run docs:dev
```

* 打包
```js
npm run docs:build
```

* 打包并上传到github
```js
npm run deploy
```
